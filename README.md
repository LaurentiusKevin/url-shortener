## Config
This project using postgresql database
- the configuration file is in config/database.js

## Installation and Testing
Run this in console / terminal
```shell
# install
npm install

# run without hot reload
npm start

# run with hot reload
npm install -g nodemon
npm run watch

# unit test
npm test
```
